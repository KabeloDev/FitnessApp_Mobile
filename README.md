<h2>Description</h2>
Fitness tracker mobile app, built with Flutter and connected to an ASP.NET Core 9 Web API backend! It includes:

ðŸ”¹ User Profiles: Secure sign-up, profile management, and authentication. 
<br/>
ðŸ”¹ Exercise Library: Detailed exercises, muscle groups, and tutorials with easy category filtering. 
<br/>
ðŸ”¹ Workout Planner: Create, edit, and track workouts, pin them to your calendar, and monitor progress.
